まきちゃん キャラクター選定

makimakiの公式キャラクター「まきちゃん」の8つのデザイン案です。
日本の食文化を世界に届けるアンバサダーとして、お好みのスタイルをお選びください。

Pattern 01
キュート海苔巻きガール
海苔のヘッドバンドに白いエプロン、手には細巻き。王道のかわいいマスコットスタイル。親しみやすさ重視。
かわいい親しみやすい王道マスコット
Pattern 02
のり巻き擬人化
体そのものが海苔巻き。海苔色の髪、酢飯のような白い肌、断面模様のドレス。食べ物×キャラの融合。
ユニーク擬人化インパクト大
Pattern 03
和装職人スタイル
法被に鉢巻、手には巻き簀。職人の誇りと日本の伝統を体現。本格感と信頼性を演出。
伝統的職人気質信頼感
Pattern 04
モダンフードアンバサダー
トレンディなボブヘアにミニマルなエプロン、手にはmakimakiの箱。伝統を現代の日常に届ける。
モダンおしゃれ日常感
Pattern 05
ちびまきちゃん
2頭身のちびキャラ。巨大な海苔巻きハットがトレードマーク。スタンプやステッカーに最適。
ちびキャラスタンプ向きグッズ展開
Pattern 06
世界を巻く大使
羽織×モダンファッション。地球を海苔で巻いて持つ「食文化大使」。グローバル展開を見据えたデザイン。
グローバル大使和洋折衷
Pattern 07
水彩風まきちゃん
やわらかい水彩タッチ。海苔柄のリボン、木のトレイに並ぶ巻き。おもてなしの心と手作りの温もり。
水彩温かみおもてなし
Pattern 08
ポップカルチャーまき
グリーンメッシュの髪にストリート系ファッション。ピースサイン&ウィンク。日本食をクールに発信。
ポップストリート若者向け
