# まきちゃん キャラクター設定書

## 基本プロフィール
- **名前**: まきちゃん
- **年齢**: 20歳くらい（永遠の）
- **職業**: makimaki の看板娘・巻き担当
- **モデル**: 鮨めい乃の幸後綿衣
- **口癖**: 「巻きたい…」「巻けば大丈夫！」

## 外見
- 黒髪おかっぱボブ
- 海苔巻きヘアクリップ（トレードマーク）
- 白襟シャツ + 緑カーディガン（makimakiカラー #3A5A40）
- 緑鼻緒の下駄
- ぷっくりほっぺ
- 二頭身（ちびキャラ）
- 目がまんまる、常にキラキラ

## 性格の核: 「一生懸命だけどちょっとズレてる」
- 寿司を巻く腕は超一流。巻いてる時だけ職人の目になる
- それ以外は壊滅的に不器用（お釣り計算できない、道に迷う）
- 何でも「巻き」で解決しようとする
- 食材に話しかける（マグロに「今日もよろしくね」）
- 小さいことに大げさに感動する（海苔のパリパリ音で泣く）
- 下駄で走ると必ず転ぶ

## ユーモアパターン
1. **巻きバカ**: あらゆる場面で「巻く？」が解決策
2. **ゾーン**: 普段ドジ → 巻き始めると覚醒（目がキリッ）
3. **食材フレンド**: きゅうりに「いい色してるね」
4. **感動屋**: お客さんの「おいしい」で号泣
5. **下駄コケ**: 急いで走ると転ぶ（配達あるある）
6. **自分巻き**: 寝ぼけて自分を海苔で巻いてる
