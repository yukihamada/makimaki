最新の注文

トグルで在庫切れの管理ができます。OFFにするとお客様のメニューに「売切」と表示されます。

Star CloudPRNT 設定

詳しいセットアップ手順はこちら →

セットアップ手順
  1. Star mC-Print3（WiFi版）をWiFiに接続
  2. プリンターの設定画面にブラウザでアクセス（http://プリンターのIP）
  3. CloudPRNT設定を開く
  4. サーバーURLに以下を入力：
    https://makimaki.tokyo/api/cloudprnt クリックでコピー
  5. ポーリング間隔を 5秒 に設定
  6. メディアタイプを text/plain に設定
  7. 上の「プリンター有効」をONにする

LINE通知設定

注文が入ったらオーナーのLINEに通知が届きます。LINE Developers Console から取得してください。

取得手順
  1. LINE Developers Console にログイン
  2. プロバイダー作成 → 「Messaging API」チャネルを作成
  3. Messaging API設定 → 「Channel access token」の「Issue」をクリック → コピー
  4. Basic設定 → 「Your user ID」をコピー（Uで始まる33文字）
  5. 上のフォームに貼り付けて「保存」
